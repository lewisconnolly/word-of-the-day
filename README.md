# Word of the Day

A minimalist web app that displays a randomly selected uncommon English word each day with its definition, pronunciation, and part of speech.

## Features

- **Daily word** — deterministic selection so all users see the same word each day
- **New Word button** — fetch a random uncommon word on demand
- **Word history** — browse previously viewed words, persisted in localStorage
- **Dark mode** — automatically follows system preference
- **Responsive** — mobile-first design, works on all screen sizes
- **Accessible** — WCAG 2.1 AA compliant with keyboard navigation and ARIA attributes

## Tech Stack

HTML5, vanilla CSS, vanilla JavaScript — no frameworks, no build tools.

Uses the [Free Dictionary API](https://dictionaryapi.dev/) (no API key required).

## Usage

Open `index.html` in a browser, or serve with any static file server.
